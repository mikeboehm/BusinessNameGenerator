BusinessNameGenerator
=====================
Generates a list of two word business names.

The generater uses Oren Tiroshâ€™s [mnemonic encoding wordlist](http://web.archive.org/web/20090918202746/http://tothink.com/mnemonic/wordlist.html) which is comprised of 1633 short words that are easily recognisable.

It produces a lot of junk, but plenty of gems!
